Jutsu State Machine (AFND)

Projeto que simula um Autômato Finito Não Determinístico (AFND) para reconhecer jutsus a partir de sequências de selos.

- Cada selo é uma entrada
- O sistema mantém múltiplos estados simultaneamente
- Reconhece diferentes jutsus com base na sequência

Entrada:
cobra dragao coelho tigre

Saída:
Katon: Goukakyuu

Conceitos utilizados
- Autômatos Finitos
- AFND (não determinismo)
- Máquina de estados
